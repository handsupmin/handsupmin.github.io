> TL;DR
- 미디어 쿼리는 CSS에서 디바이스 특성에 따른 스타일 적용 기술
- 반응형 웹 디자인에 필수적이며 화면 크기, 해상도, 방향 등 조건 지원
- @media 키워드와 조건문으로 작성하며 우선순위 관리 중요

미디어 쿼리는 CSS에서 화면 크기, 해상도, 방향 등 디바이스 특성에 따라 스타일을 다르게 적용하는 기술입니다. 주로 반응형 웹 디자인에 사용되며, 다양한 기기에서 최적화된 사용자 경험 제공에 필수적입니다. 기본 문법은 @media 키워드 뒤에 조건을 명시하고, 중괄호 안에 적용할 CSS를 작성하는 방식입니다. 예를 들어, 화면 너비가 600px 이하일 때 특정 스타일을 적용하려면 @media (max-width: 600px) { /* 스타일 */ } 형태로 작성합니다. 미디어 쿼리는 화면 너비 외에도 해상도, 방향(가로/세로), 색상 등 다양한 조건을 지원합니다. 이를 통해 모바일, 태블릿, 데스크톱 등 각기 다른 환경에 맞춘 세밀한 디자인 조정 가능. 미디어 쿼리 사용 시 주의할 점은 조건이 겹칠 때 우선순위와 상속 관계를 명확히 이해하는 것과, 너무 많은 쿼리 사용으로 코드 복잡도가 높아지지 않도록 관리하는 것임. 반응형 웹 구현 시 필수 요소로, 사용자 환경에 맞춘 유연한 레이아웃과 스타일링에 활용됨.
