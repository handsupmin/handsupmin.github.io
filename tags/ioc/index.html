NestJS ModuleRef로 런타임 의존성 주입 제어하기

개념/배경 ModuleRef는 NestJS에서 런타임에 의존성을 직접 조회·생성할 때 쓰는 유틸리티 클래스 정적 DI로 풀기 어려운 동적 흐름 제어에 사용 핵심 기능 get: 현재 모듈 컨텍스트의 정적으로 선언된 프로바이더 인스턴스 조회 resolve: 스코프를 고려해 새로운 프로바이더 인스턴스 동적 생성 create: 클래스의 새 인스턴스를 의존성 주입과 함께 동적 생성 주의 프로바이더 스코프에 따라 반환 인스턴스가 달라질 수 있음 과도한 사용은 테스트와 구성 복잡도 증가 참고자료 https://docs.nestjs.com/fundamentals/module-ref
